==============================
lb_toolkits使用说明
==============================


.. warning::

    当前 ``lb_toolkits`` 库还不是一个稳定的版本，正在对接口API
    进行测试补充，争取把接口满足大家需求。



``lb_toolkits`` 是一个支持 ``python`` 的库，并支持以下功能：


+------------------------+----------------------+--------------------------------+----------+
|         库名           |    对象              |         功能                   |   备注   |
+========================+======================+================================+==========+
|                        | downloadLandsat      | 下载LandSat TM/ETM数据         |          |
|                        +----------------------+--------------------------------+----------+
| downloadcentre         | downloadMODIS        | 下载MODIS 各类产品数据         |          |
|                        +----------------------+--------------------------------+----------+
|                        | downloadSentinel     | 下载哨兵1/2/5数据              |          |
|                        +----------------------+--------------------------------+----------+
|                        | downloadOCO          | 下载OCO-2/3数据                |          |
|                        +----------------------+--------------------------------+----------+
|                        | downloadGOSAT        | 下载GOSAT XCO2、XCH4数据       |          |
|                        +----------------------+--------------------------------+----------+
|                        | downloadCALIPSO      | 下载CALIPSO L1、L2级数据       |          |
|                        +----------------------+--------------------------------+----------+
|                        | downloadFY           | 下载FY3、FY4近实时数据         |          |
|                        +----------------------+--------------------------------+----------+
|                        | downloadH8           | 下载H8 L1 nc/hsd数据           |          |
|                        +----------------------+--------------------------------+----------+
|                        | downloadERA5         | 下载ERA5再分析资料             |          |
|                        +----------------------+--------------------------------+----------+
|                        | downloadGFS          | 下载GFS预报资料                |          |
|                        +----------------------+--------------------------------+----------+
|                        | downloadDEM          | 下载DEM 12.5/30/90米           |          |
|                        +----------------------+--------------------------------+----------+
|                        | downloadLandcover    | 下载清华发布的10米地表类型     |          |
+------------------------+----------------------+--------------------------------+----------+
| orbit                  | orbit                | 轨道预报和交叉计算             |          |
|                        +----------------------+--------------------------------+----------+
|                        |                      |                                |          |
+------------------------+----------------------+--------------------------------+----------+
