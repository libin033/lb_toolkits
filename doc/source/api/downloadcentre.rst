=================================
downloadcentre
=================================

可以对各类卫星数据下载
 * 卫星数据：MODIS、OCO-2/3、CALIPSO、GOSAT
 * FTP下载：风云3/4、葵花8/9、海洋1/2
 * spider下载: LandSat、sentinel、TANSAT
 * 再分析资料：EAR5、GFS等
