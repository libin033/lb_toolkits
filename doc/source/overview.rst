========
Overview
========

downloadcentre
-------------------
可以对各类卫星数据下载：
 * 卫星数据：MODIS、OCO-2/3、GOSAT；
 * FTP下载：风云、葵花8；
 * spider下载: LandSat、sentinel、TANSAT；
 * 再分析资料：EAR5、GFS等；


utils
-------------------
支持ftp、sftp、spider、wget等多种下载API


.. note::
    fypy是一个支持气象卫星、高分辨率卫星的数据投影、拼接、重采样等功能，并且支持大气校正、融合、正射校正等。

    fypy相关功能正在开发、测试过程中，敬候发布...
