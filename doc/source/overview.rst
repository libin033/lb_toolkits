========
Overview
========

downloadcentre
-------------------
可以对各类卫星数据下载：
 * 卫星数据：MODIS、OCO-2/3、GOSAT；
 * FTP下载：风云、葵花8；
 * spider下载: LandSat、sentinel、TANSAT；
 * 再分析资料：EAR5、GFS等；


utils
-------------------
对hdf4、hdf5、netcdf4、geotiff等格式文件进行读写操作；

calc
-------------------
提供了气象、遥感方向的各类计算公式；

    fypy相关功能正在开发、测试过程中，敬候发布...
